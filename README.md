# Todo List

### Using Custom Webpack
- node version: 18.14.2
**Give it a ⭐ if you liked it 😜**
# demo
see demo [here](https://todo-list-drag-and-drop.vercel.app/)



## install

```bash
npm i
```

## start app

```bash
npm start
```

**_Tech used_**

- react
- tailwindcss

## Features

- Add Todo
- Remove Todo
- Edit todo bg click
- Drag and Drop todo
- Add multi line task 
- Auto save to local storage
- Each line is considered as a todo
